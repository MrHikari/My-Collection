## Sass (Syntactically Awesome StyleSheets)

**Sass** 是一款**强化** *CSS* 的辅助工具。<br/>
在 *CSS* 语法的基础上增加了
* 变量 (variables)
* 嵌套 (nested rules)
* 混合 (mixins)
* 导入 (inline imports) 
* 等高级功能，

使用 **Sass** 以及 Sass 的**样式库**有助于更好地组织管理样式文件，更高效地开发项目。

### 特征 (Features)

* 完全兼容 CSS3
* 在 CSS 基础上增加变量、嵌套 (nesting)、混合 (mixins) 等功能
* 通过函数进行颜色值与属性值的运算
* 提供控制指令 (control directives)等高级功能
* 自定义输出格式
